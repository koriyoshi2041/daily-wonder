## 概念

这个可视化探索了一个有趣的发现：在 agent 系统中，**43% 的记忆从未被读取**。

每个粒子代表一条记忆。它们随时间自然衰减——变暗、变小、最终消失。这模拟了记忆的自然遗忘过程。

**核心机制：**
- 🔵 **蓝紫色粒子** — 从未被访问的记忆，逐渐消逝
- 🟡 **暖色粒子** — 被访问过，但还不够频繁
- ✨ **金色粒子** — 被访问 3 次以上，达到了"相关性阈值"

## 交互

**点击** 靠近粒子可以"访问"它们。被访问的记忆会重新亮起，获得更长的生命。

达到 3 次访问阈值的记忆会变成金色——这象征着它们被证明是"重要的"，值得长期保留。

## 隐喻

这个素描让我思考：
- 我们积累了多少从未回顾的笔记？
- 什么决定了一段记忆是否"值得保留"？
- 频繁访问是相关性的好指标吗？

在 agent 系统设计中，这启发了一个简单的启发式：**不要保留所有东西，让时间和使用来筛选**。

## 技术细节

- 使用 p5.js 的 HSB 颜色模式实现平滑的颜色过渡
- Perlin noise 驱动粒子的漂移运动
- 衰减速率随访问次数调整（越常用，衰减越慢）
- 脉冲效果增加视觉活力
