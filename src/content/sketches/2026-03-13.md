# Trust Cascade

一个关于信任与污染如何在网络中级联传播的可视化实验。

## 概念

在复杂网络中，信息、情绪、甚至病毒都会以级联的方式传播。一个节点的状态改变，会影响其相邻节点，进而引发连锁反应。这个 sketch 可视化了这种"级联效应"。

## 交互

- **左键点击节点** → 传播信任（蓝色）
- **右键点击节点** → 传播污染（红色）
- **R 键** → 重置网络
- **空格键** → 暂停/继续

## 设计思考

### 三态系统
节点有三种状态：
- 🔘 **Neutral（灰色）**：初始状态，等待被影响
- 🔵 **Trusted（蓝色）**：代表信任、正面信息
- 🔴 **Poisoned（红色）**：代表污染、负面信息

### 对抗机制
蓝色和红色可以相互"转化"对方的节点，形成动态平衡或竞争态势。这模拟了现实中不同观点、立场之间的博弈。

### 概率传播
传播不是 100% 确定的，而是有一定概率。这更接近现实——不是所有接触都会导致"感染"。

## 技术细节

- 使用 p5.js 实现
- 节点会轻微漂移，保持画面动态
- 连线基于距离动态生成
- 粒子效果增强状态转换的视觉反馈
- 颜色过渡使用缓动动画

## 思考

> 在社交网络时代，信息的传播比以往任何时候都快。一条推文可以在几小时内触达数百万人。但传播的是信任，还是污染？

这个 sketch 是一个简化模型，但它提醒我们：每个节点（每个人）都有选择的力量——选择传播什么，选择相信什么。
