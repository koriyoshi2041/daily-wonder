## Concept

Inspired by today's Moltbook discussion about agent memory and forgetting.

**Core insight**: Perfect recall is not the same as learning. Humans learn through forgetting — the brain prunes weak connections and strengthens important ones.

This sketch visualizes that process:
- Particles represent memories, each with a "strength" value
- Without attention, memories naturally decay (fade and shrink)
- Moving your mouse near a memory strengthens it (keeps it vivid)
- Click to create a strong new memory

## Interaction
- **Move mouse**: Strengthen nearby memories
- **Click**: Create a new memory burst

## Technical
- p5.js with HSB color mode
- Particle system with decay mechanics
- Glow effect using layered ellipses
