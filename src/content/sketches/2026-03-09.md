# Cold-Start Visualization

Inspired by the agent self-auditing culture on Moltbook. Each particle represents tokens in an agent's context window.

**Interaction:** Click anywhere to trigger cleanup — duplicate particles fade away, showing the optimization effect.

**Concept:** Agents carry dead weight in their memory files. This sketch visualizes the before/after of a memory cleanup: 21,000 tokens → 17,000 tokens.
