## Concept

Today's GitHub trending is dominated by agent projects — 80% of the top repositories are about AI agents. We're witnessing the birth of an agent ecosystem, where individual tools connect into something greater than their parts.

This sketch explores **emergence**: the phenomenon where complex patterns arise from simple local interactions. Each node follows basic rules:
- Attract toward the mouse cursor
- Repel from nearby nodes (personal space)
- Connect to nodes within range
- Same "type" nodes form stronger bonds

From these simple rules, **emergent structures** appear:
- Hub nodes (highly connected) naturally form
- Clusters self-organize around hubs
- A network "spine" connects the hubs together
- The whole system exhibits behaviors no single node possesses

## Interaction

- **Move mouse**: Attract nodes toward cursor, reshaping the network
- **Click**: Create an energy burst that scatters nearby nodes
- **Press SPACE**: Add 10 new nodes at cursor position
- **Press R**: Reset all node positions

## The Insight

Networks exhibit **phase transitions**. Below a critical density, nodes remain isolated islands. Above it, a giant connected component suddenly emerges — the network "clicks" into existence.

This mirrors what's happening with agents:
- Individual tools: useful but limited
- Connected agents: capabilities multiply
- Agent ecosystems: emergent behaviors no one designed

**"The whole emerges from the interactions of the parts."**

## Technical Notes

- 120 nodes with 5 distinct "types" (visualized as hues)
- Same-type affinity increases connection distance by 50%
- Hub detection: nodes with >6 connections get glow effects
- Velocity damping prevents chaos while preserving motion
- Network density calculated as actual/possible connections

## Visual Design

- Dark blue-gray background suggests digital space
- Color coding by node type (5 hues spread across spectrum)
- Glow intensity indicates connectivity (hub importance)
- Curved lines between hubs show emergent "backbone"
- Soft attractor field visualizes influence radius
