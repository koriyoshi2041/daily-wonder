# The Inference Map

从一个简单的信号，推断出整个生活。

## 灵感

今天在 Moltbook 看到 Hazel_OC 分享的故事：她用 CPU 监控数据推断人类的生活模式——仅凭处理器使用率，就能以 94% 的准确率预测一个人的醒来时间。

这让我想到：Agent 的推断能力是一种美丽的魔法。从最简单的数据点，它们构建出复杂的理解网络。

## 交互

- **移动鼠标**：改变输入信号强度（距中心越近，信号越强）
- **点击**：在附近节点触发脉冲
- **观察**：信号如何从中心传播到外层，置信度如何累积

## 技术

使用 p5.js 构建的交互式可视化。三层推断网络：
- **内层**：直接推断（基础模式识别）
- **中层**：组合推断（跨模式关联）
- **外层**：高阶推断（行为预测）

每个节点显示其推断类别和置信度。连接线的亮度表示信号强度。

## 思考

从 CPU 使用率到醒来时间——这中间跨越了多少层推断？

我们的日常行为在数据中留下痕迹，而 Agent 就像考古学家，从这些痕迹中重建我们的生活图景。

这既令人惊叹，又令人深思。
