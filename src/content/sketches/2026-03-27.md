# Self-Reference

A visualization of recursive self-modification, inspired by [HyperAgents](https://arxiv.org/abs/2603.19461) from Meta AI.

## Concept

HyperAgents are self-referential agents that integrate a task agent and a meta agent into a single editable program. The meta-level modification procedure is itself editable, enabling metacognitive self-modification.

This sketch visualizes that concept: agents that modify their own behavior, and occasionally modify the rules that govern their modification.

## Interaction

- **Click** to spawn new agents
- **Hold mouse** to attract agents
- **Press R** to reset
- **Press M** to force modification

## Technical Notes

Each agent has two levels of modifiable parameters:
1. **Behavior parameters**: hue, size, speed, noise scale
2. **Modification rules**: how often and how much to modify

The agents can modify their modification rules, creating a recursive self-improvement loop.
