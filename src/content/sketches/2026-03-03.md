# Recovery/Rollback

**Visualizing the power of undo — how agents can gracefully recover from any state.**

## Concept

In agent systems, the ability to checkpoint and rollback is fundamental to reliability. This sketch transforms that abstract concept into a visceral, interactive experience.

Particles drift freely through space, influenced by your mouse cursor. At any moment, you can create a **checkpoint** — freezing the current state in time. When you **rollback**, the particles smoothly animate back to their saved positions, demonstrating graceful state recovery.

## Interactions

- **Mouse movement**: Attracts nearby particles
- **Click**: Creates a burst of energy towards the cursor
- **SPACE**: Create a checkpoint (saves current particle positions)
- **R**: Rollback to the last checkpoint

## Technical Details

Each particle maintains:
- Position and velocity state
- Visual trail history
- Target position for smooth rollback animation

Checkpoints store the complete state of all particles. The rollback uses linear interpolation (lerp) to create smooth transitions — particles don't just teleport back, they *flow* through space to their saved positions.

## Why This Matters

This isn't just a pretty animation. It's a metaphor for:
- **Version control** in code
- **Undo systems** in creative tools
- **State management** in applications
- **Recovery mechanisms** in distributed systems

The visual feedback makes the abstract tangible — you can *see* what it means to save and restore state.
